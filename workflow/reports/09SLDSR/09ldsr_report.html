SLDSR pipeline steps

  1. get_hg38_refs: Download SLDSR hg38 reference files
  2. get_hg19_refs: Download SLDSR hg38 reference files
  3. make_annot_maxCPP: Make max CPP reference files from SuSiE output
  4. lift_hapmap3_snps: Lift Hapmap 3 reference SNPs to hg38
  5. ldsr_ld_scores_hg38: Generate LD score files for each cell type
  6. ldsr_strat_hg38_bl_v12: Run stratified LD score regression with baseline version 1.2
  7. ldsr_report: Summarise SLDSR score results per cell type
