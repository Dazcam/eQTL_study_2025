Prepare GWAS

Overview

The S-LDSR pipeline quantifies the enrichment of heritability for specific GWAS traits within the functional regions identified by our eQTL mapping. Specifically, it uses the maxCPP annotation from the SuSiE fine-mapping output to determine if cell-type-specific eQTLs are enriched for the heritability of neuropsychiatric traits.

Workflow Logic

The pipeline follows a tiered execution strategy to manage heavy I/O and storage constraints.

  1. get_hg38_refs: Download SLDSR hg38 reference files.
  2. get_hg19_refs: Download SLDSR hg38 reference files.
  3. make_annot_maxCPP: Converts SuSiE credible sets into LDSR-compatible MaxCPP annotation files.
  4. lift_hapmap3_snps: Lift Hapmap 3 reference SNPs to hg38 coordinates.
  5. ldsr_ld_scores_hg38: Calculates stratified LD scores for each cell type (MaxCPP anns).
  6. ldsr_strat_hg38_bl_v12: Run stratified LD score regression (partitioned by cell type and trait) with baseline version 1.2.
  7. ldsr_report: Aggregates all enrichment scores into a finalized HTML report.
Note

Coordinate Consistency

LDSR was originally built using references for hg19. However, hg38 references are also avaliable and we used the latter for our analysis. Note GWA aligned to hg19 were lifted over to hg38 in the Prep GWAS pepeline.

Technical requirements

Category Requirement
Software LDSR (LD Score Regression v1.0.1)
Environment ldsr (Conda) / r_eqtl (Singularity)
Reference Panels 1000 Genomes hg38 (Baseline v1.2)
Functional Inputs SuSiE maxCPP annotations
SNP Set HapMap3 (Lifts to hg38 via CrossMap)

Implementation Detail

The pipeline uses eval to hook the Conda environment within the shell block. This ensures that the ldsc Python 2.7 environment (required by the original software) is correctly isolated from the Snakemake execution environment.

# Environment Activation Pattern
eval "$(/apps/languages/miniforge3/24.3.0-0/bin/conda shell.bash hook)"
conda activate ldsr

Run Composition & Throughput

The S-LDSC pipeline iterates across chromosomes, cell types, functional annotations, and GWAS traits. Although resource requirements are minimal I used the qos-500 facility on Hawk to get all the jobs through quickly.

  • Cell Types: 19 lineages (including sub-clusters like Glu-UL-0/1/2, GABA-0/1/2, etc.)
  • Functional Annotations: MaxCPP
  • Chromosomes: 22 autosomes
  • GWAS Traits: 5 neuropsychiatric disorders
Phase Total Jobs Complexity
make_annot_maxCPP 418 19 cell types X 1 Annotation X 22 Chrs
ldsr_ld_scores_hg38 418 Same as annotation phase above
ldsr_strat_hg38_bl_v12 95 19 cell types X 1 Annotation X 5 GWAS
